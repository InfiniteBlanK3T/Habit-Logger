# Habit Logger

Practice my C# skill and CRUD, SQL against a real database, using VS2022, ADO.NET. 

## Description

- Create a database to store my running record(default/ user can create a new record if they want to).
- If you want to check out my code feel free to!
- The database is stored in [Your project folder]\bin\Debug\net[version]\[file].db
## Requirements
- [x] When the application starts, it should create an SQLite database if one isn’t present.
- [x] Application should store and retrieve data from a real database
- [x] Users can be able to insert, delete, update, and view their logged habit.
- [x] Users can be able to create a new SQLite database for their own habit.
- [x] Report functionality based on the given information in the database.
- [x] Handle all possible errors / Did run unit tests all passed *so far*.
- [x] Interact with the database using raw SQL. Cannot use mappers such as [Entity Framework](https://learn.microsoft.com/en-us/ef/core/get-started/overview/first-app?tabs=netcore-cli).
- [x] README file.
